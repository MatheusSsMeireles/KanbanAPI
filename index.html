Entendendo as APIs

Bem-vindo ao Chatwoot Dev Explorer. Esta ferramenta foi projetada para que você possa entender e testar as capacidades do seu Chatwoot de forma centralizada.

A API Kanban (Extensão)

É a documentação customizada que aborda a gestão de Funis de Vendas, movimentação de Cards (Kanban Items), Checklist, Notas, e Mensagens Agendadas. Você testará essa API diretamente aqui.

A API Oficial

Trata do "Core" do sistema: criação de contatos, caixas de entrada (Inboxes), troca de mensagens nativas e relatórios gerais. Direcionamos você à fonte oficial para estas consultas.

Segurança 100% Local: Suas credenciais (URL e Token) nunca são enviadas para nenhum servidor externo. Elas são manipuladas exclusivamente pelo seu navegador (Client-Side) para realizar as chamadas diretas para a sua própria instância do Chatwoot.
